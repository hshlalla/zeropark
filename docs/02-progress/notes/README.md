# 개발 노트

이 폴더는 개발 중 생기는 작업 노트를 남기는 곳입니다.

## 언제 작성하나

- LLM과 함께 구현하면서 중요한 판단을 했을 때
- 시도했다가 포기한 접근이 있을 때
- 임시 우회나 기술 부채가 생겼을 때
- 블로커, 리스크, 확인 필요 사항이 생겼을 때
- 다음 작업자가 이어받아야 할 맥락이 있을 때

## 파일명 규칙

```text
YYYY-MM-DD-note-title.md
YYYY-WW-weekly-summary.md
DEV-001-gitlab-commit-api.md
```

## 주의사항

secret, 토큰, 비밀번호, 내부 접속 정보는 작성하지 않습니다.
